blksc3
------

Block SuperCollider,
or SuperBlock,
is a [Block](https://developers.google.com/blockly) editor for the
[SuperCollider](https://www.audiosynth.com/) synthesiser.

![](https://rohandrape.net/sw/blksc3/svg/BlockSuperCollider.svg)
