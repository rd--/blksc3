# blksc3

Block SuperCollider.

## blk-gen

Generate `.js` and `.json` and `.xml` files for blksc3.

````
$ blksc3 blk-gen
...
$
````

## stc-to-osc

Read .stc text notation (in fact any text) from a Websocket and send it as the single parameter of an `/eval` OpenSoundControl message.

````
$ blksc3 stc-to-osc
...
$
````
